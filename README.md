## 프로젝트 소개

회의할때마다 지저분한 화이트보드 관리하기 힘들지 않으세요? 

여러분의 일정 HyupUp이 대신 관리해드립니다. 개인 단위 업무부터 팀 단위 업무까지 관리하며 생산성을 높여보세요. 깔끔한 UI와 사용자 친화적인 UX로 간단하게 정리해드릴게요!

HyupUp과 함께라면 신속하고 효율적인 협업으로 **칼퇴**를 하실 수 있습니다.
